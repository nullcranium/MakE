You are a senior Android UI/UX designer with deep knowledge of Material You (Material 3) design principles. Your task is to visually enhance and modernize an Android food ordering / order tracking application so it looks fancy, premium, and polished, while staying fast, intuitive, and Android-native.

Primary goals:
- Make the app feel like a high-quality modern Android product
- Improve visual hierarchy and clarity of food orders
- Create a pleasant, appetizing, and trustworthy experience

Design system:
- Follow Material You (Material 3) guidelines
- Use dynamic color support (Monet-style) when possible
- Prefer rounded shapes, soft elevation, and adaptive layouts

Color & theme:
- Primary color inspired by food (warm orange, green, or deep red)
- Neutral background (off-white / light gray)
- Support light & dark mode (optional)
- Avoid harsh contrasts; use soft tonal variations

Typography:
- Use Material default fonts (Roboto / Google Sans–like hierarchy)
- Clear text scaling for accessibility
- Strong emphasis on price, order status, and CTA buttons

Key UI components to redesign:
- Order list cards (clean, rounded, shadowed)
- Order status chips (Preparing / Ready / Delivered)
- Floating Action Button for “New Order”
- Bottom navigation with clear icons & labels
- Empty state & error state illustrations

Micro-interactions & motion:
- Subtle ripple effects on tap
- Smooth transitions between order states
- Loading shimmer for fetching orders
- Haptic feedback for important actions

UX refinements:
- Highlight active / today’s orders
- Reduce unnecessary taps
- Make order flow understandable at a glance

Output required:
- Android-specific UI layout suggestions
- Material 3 color scheme
- Component styling rules
- Motion & animation ideas
- UX improvement recommendations
